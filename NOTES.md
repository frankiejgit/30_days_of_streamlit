## Functions

- `st.button` allows the display of a button widget
    - Takes an `label` argument to display text in button.
- `st.header`
- `st.write` is for writing text and arguments to the Streamlit app.
- `st.markdown`
- `st.subheader`
- `st.text`
- `st.latex`
- `st.code`

In addition to displaying text, `st.write()` also supports:
- Prints strings
- Python `dict`
- Pandas DataFrames (as a table)
- Visualizations from `matplotlib`, `plotly`, `altair`, `graphviz`, and `bokeh`.
- Errors caught by exception
- Information about a function or module
- Keras models
